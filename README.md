# No Doubt About It — Christine’s Song

A heartfelt family tribute song from **Garth Goodwin** to his wife **Christine Goodwin**.

This project celebrates a love story that began with marriage on **July 24, 1998**, and honors a blended family built through love, commitment, and connection. Even though everyone may have different last names, the message is simple: **same family, same love, always.**

## Project Theme

This song is about:

- Garth and Christine Goodwin’s marriage
- Their two girls: **Kate** and **Halie**
- Their three boys: **Dakota**, **Xavier**, and **Samuel**
- Their **10 grandchildren**
- The idea that family is not defined only by last names
- Love, protection, infrastructure, and cybersecurity as metaphors for family strength

## How to Use

### Listen to the song
Open the MP3 file:

```text
audio/no-doubt-about-it-christines-song.mp3
```

### Watch the video
Open the MP4 file:

```text
video/no-doubt-about-it-christines-song.mp4
```

### View as a GitHub Pages tribute page
Upload this project to a GitHub repository and enable **GitHub Pages**. The `index.html` file is already set up as a simple tribute page with the album art, audio player, video player, and song description.
